# Especificação de Requisitos

## 1. Requisitos Funcionais (RF)
| ID | Descrição | Prioridade |
|:---|:---|:---|
| RF-001 | Login seguro via E-SEMEC | Essencial |
| RF-007 | Lançamento de frequência/faltas | Essencial |
| RF-013 | Registro de conteúdo ministrado | Essencial |
| RF-030 | Funcionamento em Modo Offline | Crítico |
| RF-011 | Alertas de limite de faltas | Desejável |

## 2. Requisitos Não-Funcionais (RNF)
- **RNF-001:** Disponível para Android (8.0+) e iOS (12.0+).
- **RNF-010:** Banco de dados PostgreSQL.
- **RNF-015:** Criptografia TLS 1.2+ em todos os dados em trânsito.
- **RNF-044:** Conformidade total com a LGPD.

*(Para a lista completa, consulte o Documento de Requisitos v1.0 original)*
